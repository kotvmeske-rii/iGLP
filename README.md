# iGLP: Parser and Solver for Provability Logics

## Roadmap
`GL` ➔ `iGL` ➔ `GLP` ➔ `iGLP`

---

## Связь с научной работой
Моя научная сфера интересов лежит в области теории доказуемости. Текущая задача - сформулировать интуиционистский аналог полимодальной логики доказуемости Джапаридзе $(iGLP)$. Для этого необходимо проверять чтобы вводимые аксиомы были общезначимы. В нашем солвере нужно проверять, существуют ли контрпримеры — именно это код и делает. Сейчас код на стадии перехода из прувера для $GL$ к пруверу для $iGL$

---

## Структура проекта
* `/parser` — Lexer, parser, and AST definition
* `/solver` — Kripke models, worlds evaluation, and frame logic
